## 插件用法概览

### 1. Credit信用点主插件
*   **主命令：** `/credit` 可以查看用法
*   **说明：** 主插件附带复活台物品。
*   **附属插件：** 监管，交换器，借记单，地狱控制器。

### 2. 竞技场插件
*   **前置需求：** Credit信用点主插件
*   **主命令：** `/arena` 可以查看用法
*   **附属插件：** 无

### 3. 玩家刺客插件
*   **前置需求：** Credit信用点主插件
*   **主命令：** `/hire` 查看用法
*   **附属插件：** 无

### 4. 怪物刺客插件
*   **前置需求：** Credit信用点主插件
*   **主命令：** `/assasin` 查看用法
*   **附属插件：** 无

### 5. 溺尸王（桑恩）插件
*   **主命令：** `/drowned` 查看用法
*   **附属插件：** 太平洋之风插件，死亡之冠插件

### 6. 暴君（卢纳）插件
